# 🔥 La Cabaña Mar & Grill — Menú Interactivo

Menú digital interactivo del restaurante **La Cabaña Mar & Grill** en Santo Domingo, Ecuador.

Sitio estático (HTML/CSS/JS puro), desplegado en GitHub Pages.

## Features
- 🎨 Diseño dark con acentos dorados
- 🔍 Búsqueda instantánea (⌘K / Ctrl+K)
- 📂 Navegación por tabs y sub-categorías
- 💬 Pedido directo por WhatsApp en cada platillo
- 📱 100% responsive
- ✨ Animaciones suaves
